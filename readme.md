Rajesh Sakhre – Author Portfolio Website

A modern, responsive author portfolio website built using HTML, CSS, and JavaScript. This website showcases the books, biography, consulting services, and contact information of Rajesh Sakhre, an author and business consultant.

📖 About the Project

This website is designed to provide readers and visitors with:

Information about the author
Book showcase with carousel slider
Business consulting services
Inspirational quotes
Contact details for book purchases and inquiries

The design focuses on readability, elegant typography, smooth animations, and a professional presentation.
